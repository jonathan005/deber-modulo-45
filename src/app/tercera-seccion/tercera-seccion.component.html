Práctica de Angular

Para la práctica de Angular se realizó las actividades vistas en la sección 1 y sección 2. Las actividades incluyen lo siguiente:
  • Generar un nuevo proyecto con routing
  • Seleccionar el estilo a usar Angular Material o Bootstrap
  • Creación de una nueva página para realizar un ruteo simple
  • Definir las rutas en el app-routing.module.ts
  • Agregar las rutas en el DOM
  • Crear componentes para intercambiar datos entre padres e hijos
Dos sesiones no pude asistir, pero el contendio de las unidades se encuentra de forma práctica en este deber.
