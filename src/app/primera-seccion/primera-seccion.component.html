Navegación en Angular

Antes de generar un nuevo proyecto, se recomienda hacerlo con el siguiente comando:
ng new routing-app --routing
Angular Routing nos va a permitir desplazarnos entre componentes/páginas, por lo que una vez creado el proyecto, se procede a crear otro componente:
ng g component primera-seccion
Una vez que tengamos creado nuestro(s) componentes(s) nos dirigimos a app-routing-module e importamos nuestro(s) componente(s).
import {{'{'}} PrimeraSeccionComponent {{'}'}} from './primera-seccion/primera-seccion.component';
Y luego creamos una constante routes de tipo Route que contendrá el path y la referencia de los componentes a usar
const routes: Routes = [
{{'{'}} path: '', redirectTo: 'home', pathMatch: 'full'{{'}'}},
{{'{'}} path: 'home', component:HomeComponent{{'}'}},
{{'{'}} path: 'primera-seccion', component: PrimeraSeccionComponent {{'}'}},
{{'{'}} path: 'segunda-seccion', component: SegundaSeccionComponent {{'}'}},
{{'{'}} path: 'tercera-seccion', component: TerceraSeccionComponent {{'}'}},
];
